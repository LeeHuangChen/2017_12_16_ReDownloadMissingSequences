Written by Lee Chen in 2017

To run the code type the following in a UNIX terminal:

python run.py

You need the following software installed:
	- Python 2.7
	- Biopython 


Input:
	- the input folder is "PFamMappings" (configurable)
	- the input file can be downloaded from the PFam webcite (http://pfam.xfam.org/)
		- I downloaded from this link:
			- "ftp://ftp.ebi.ac.uk/pub/databases/Pfam/mappings/"
	- the format of the file should be:
		PDB_ID	CHAIN_ID	PdbResNumStart	PdbResNumEnd	PFAM_ACC	PFAM_Name	PFAM_desc	eValue
